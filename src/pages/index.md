[CWGI](https://github.com/jw-12138/cwgi-cli) is a blog comment system based on GitHub Issues with minimalism in mind.

## Features

- Fast to load (~ 25Kb after gzip)
- Naturally Safe
  - XSS free
  - Easy notifications
  - Easy management / moderation
